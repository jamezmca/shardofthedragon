How to use this generator

  • Tick the worksheet types you want and set a count for each.
  • For Venn, Tree and Table worksheets, choose a difficulty level: Beginner = diagram fully filled in (just answer questions), Intermediate = some cells missing, Advanced = all diagram values blank (fill from sentences).
  • Click Generate Worksheet to create the worksheet. Click again to regenerate with new numbers.
  • Use Print / Save PDF to print or export. In the print dialog, choose "Save as PDF" for a PDF download.
  • Questions include: basic probability, OR, AND (with/without replacement), and NOT questions.
  • Contexts shuffle each time — bags, lollies, marbles, spinners, coins, dice, tiles, fruit, lucky dips, cards.
